# 控制状态完整实现

## 修改任务登记

- 修改目标：silence/taunt/confuse/charm/fear 产生实际游戏效果
- 问题现象：这5种状态附加后无任何行为限制，等同于装饰
- 预期结果：silence禁用MP技能，taunt强制攻击来源，confuse/charm改变攻击目标，fear有概率跳过
- 非目标：不新增状态类型，不改状态附加逻辑
- 涉及文件：`commands.ts`、`ai.ts`
- 影响范围：命令校验、AI目标选择与行动决策
- 风险点：目标判定变更，需确认不影响正常技能流程

## 行为变更记录

| 状态 | 旧行为 | 新行为 |
|------|--------|--------|
| silence | 无效果 | 禁止使用 MP>0 的技能 |
| taunt | 无效果 | 强制攻击嘲讽来源单位 |
| confuse | 无效果 | 50%概率攻击己方随机目标 |
| charm | 无效果 | 强制攻击己方血量最高目标 |
| fear | 无效果 | 30%概率跳过行动（防御代替） |
