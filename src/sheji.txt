<design_天香美人图_全局交互与外层样式>
全局环境与容器 (Global Environment & Wrapper):
  遮罩层 (Overlay Backdrop):
    - 状态: 当卷轴展开时触发。
    - 视觉表现: 整个网页背景覆盖一层带玻璃拟态的暗红色遮罩 (backdrop-filter: blur(8px) brightness(0.6); background: rgba(45, 26, 30, 0.4))。
    - 氛围特效: 遮罩层上使用绝对定位的 CSS 粒子动画 (或轻量级 Canvas)，模拟极其缓慢飘落的粉色桃花瓣和升腾的“桃花瘴气”(mix-blend-mode: screen)。让整个屏幕瞬间进入一种闭塞、暧昧的“双修密室”氛围。
  全局光影 (Global Lighting):
    - 光源: 设定一个隐藏的底部光源 (box-shadow 模拟)，当卷轴打开时，画纸底部边缘会泛着微弱的莹润粉光，就像是某种欲望在烘烤着这幅画。
  鼠标指针 (Custom Cursor):
    - 当鼠标进入画卷区域时，默认指针替换为一个极小的、精致的“金红色合欢铃” SVG 图标。点击时触发短暂的缩放震动动画。

收起态设计 (Idle & Hover State - 屏幕底部):
  位置与尺寸:
    - 绝对定位在视口 (Viewport) 正下方居中 (bottom: -10px; transform: translateX(-50%))，故意露出一半，像是在勾引人去拉扯。
  视觉材质 (The Rolled Scroll):
    - 画轴卷起的状态，材质为油润的紫檀木，带有 CSS 光泽反光 (linear-gradient 模拟圆柱体的金属/木质高光)。
    - 中间束缚着一根鲜红色的丝带 (SVG 绘制)，打着一个活结。
  交互与微动效 (Micro-interactions):
    - 待机呼吸 (Breathing): 卷轴整体有一个极其微弱、缓慢的上下浮动 (transform: translateY 波动，周期 4s)，模拟一种活物的呼吸感。
    - 悬停撩拨 (Hover): 当鼠标悬停在卷轴上时，触发以下联动：
      1. 红色丝带的尾端开始像被微风吹动一样轻轻摇曳 (CSS transform: rotate + skew 动画)。
      2. 卷轴周围渗出一点点粉红色的外发光 (box-shadow: 0 0 15px rgba(220, 80, 110, 0.6))。
      3. 稍微向上浮出水面 (bottom: 0px)，暴露出完整的轴体，等待被“解开”。

展开动画流 (Opening Sequence - 宽衣解带隐喻):
  触发机制: 点击收起态的卷轴。
  动画分步执行 (CSS Keyframes & setTimeout 编排):
    - Step 1 (解带): 绑在卷轴上的红色丝带活结散开，丝带变成两条线向屏幕下方滑落并淡出 (opacity: 0, transform: translateY(50px))。
    - Step 2 (升腾): 卷轴本体平滑地上升到屏幕正中央 (top: 50%; transform: translate(-50%, -50%))，运动曲线使用带有弹性的缓动函数 (cubic-bezier(0.34, 1.56, 0.64, 1))。
    - Step 3 (舒展): 卷轴向左右两侧拉开。这不是简单的 width 变大，而是：
      两侧的轴头 (玉柄) 向左右平移，中间的画纸 (桃花笺) 从中心向两侧展开。
      画纸展开的过程中，带有短暂的垂直方向的抖动 (类似用力抖开布料的物理效果)。
    - Step 4 (显影): 画纸完全展开后，背景的“春宫交媾图暗纹”先通过 opacity 从 0 渐变到 0.15 浮现，接着内部的四大模块内容通过 filter: blur(10px) 到 blur(0) 加上 fade-in 效果依次显现。

全局导航机制 (Navigation System - 模块切换):
  导航栏位置:
    - 位于展开画卷的右侧边缘，悬浮在画轴玉柄的内侧。
  导航样式 (印章式 Tab):
    - 四个模块【吾身】、【幻卷】、【芥子】、【群芳】不直接把名字写出来，而是设计成四枚竖向排列的“朱砂印章”或“玉牌悬挂”。
    - 未选中状态: 颜色暗淡，呈干涸的暗红色。
    - 选中状态: 印章发出鲜艳的血红色光泽，带有水润的反光特效，并在印章旁伸出一条极细的红线连接到内容区。
  模块切换转场动画 (Transition Effects):
    - 告别常规的瞬间切换或简单的左右滑动。这里采用“肉色轻纱拂过”的隐喻。
    - 当点击另一个印章时：
      1. 当前模块的内容迅速失去颜色 (filter: grayscale(100%)) 并向下溶解 (transform: translateY(20px); opacity: 0)。
      2. 一层半透明的粉红色 SVG 遮罩 (形如被风吹动的轻纱) 从右向左扫过整个画卷内容区。
      3. 轻纱扫过的同时，新模块的内容从上方平滑降落并显现 (transform: translateY(-20px) -> 0; opacity: 1)。
      4. 整个过程耗时约 0.6 秒，流畅且充满色气。

关闭/收拢机制 (Closing Sequence):
  触发区域:
    - 整个屏幕遮罩层的空白区域 (点击画卷外)，或者画卷右上角的一个“合”字小印章。
  动画分步:
    - Step 1 (内容消散): 内部的四大模块瞬间化为粉红色的墨迹粒子散开 (CSS 配合轻量动画库，或直接使用 opacity:0 加 blur(20px))。
    - Step 2 (卷起): 两侧的轴头迅速向中间合拢，画纸宽度收缩至 0。
    - Step 3 (坠落): 合拢的卷轴失去浮力，掉落回屏幕底部的初始位置，红丝带重新缠绕成结，遮罩层和桃花瘴气淡出，回到待机状态。

模块一: 【吾身】(主角信息面板)
总体布局: 采用 Flexbox 左右分栏布局。背景使用径向渐变 (radial-gradient)，中心微微泛着如肌肤般的肉粉色微光，向四周暗沉为紫檀木色。

1. 左侧栏: 宗主玉牌 (基础履历区)
   - 容器结构: 一个固定宽度的长方形卡片，边缘带有极细的 1px 金色实线边框 (border: 1px solid #D4AF37)。
   - 材质与滤镜:
     使用 CSS 玻璃拟态 (backdrop-filter: blur(10px))，背景色为带 30% 透明度的暗红色 (rgba(139, 38, 38, 0.3))，模拟滴血翡翠的质感。
     卡片内部右上角挂着一根绝对定位 (position: absolute) 的“红线”，红线使用 SVG 绘制，自然垂下。
   - 文本排版:
     姓名和身份文本居中，纵向排列 (writing-mode: vertical-rl)。字体使用带发光文字阴影的瘦金体 (text-shadow: 0 0 5px rgba(230, 106, 138, 0.8))。
   - 交互动画:
     鼠标悬停 (hover) 玉牌时，触发一个慢速的 CSS 亮度脉冲动画 (filter: brightness)，同时红线 SVG 会触发微小的摇摆动画 (transform: rotate)，像是被呼吸吹动。

2. 右侧栏: 修真面板 (数值与技能区)
   - 布局: CSS Grid 网格布局，将五维图、功法列表、法器列表错落排列。
   - 五维阵图 (六芒星雷达图):
     - 表现: 雷达图的背景网格线不是直的直线，而是带有一点弧度的血红色蛛网状 SVG 路径。
     - 填充区域: 雷达图多边形的填充色为半透明的胭脂红，边缘线带有外发光 (box-shadow/drop-shadow)。
     - 动画: 多边形的各个顶点数据在初始化加载时，会有一个从中心点向外扩散的弹性过渡动画 (cubic-bezier 缓动函数)。
   - 功法与法器列表:
     - 列表项结构: 无序列表 (ul/li)。每一项前没有传统的圆点，而是一个极小的粉色花瓣图标。
     - 文本高亮: “合欢宗双修功法”等特殊词汇，文本颜色设为高亮红，并在其下方加一层很细的、从左到右滑过的流光渐变背景动画 (background-position animation)，模拟真气流转。
     - 滚动条: 右侧的滚动条 (::-webkit-scrollbar) 被自定义为极细的暗红色线条，滑块是一个小巧的金玉色圆角矩形。

模块二: 【幻卷】(法宝监控面板)
总体布局: 居中对齐的主视觉区，周围环绕着半透明的数据仪表盘。

1. 背景特效: 淫靡雾气
   - 视觉表现: 在该模块的最底层，叠加两层绝对定位的 div。使用带有粉色云雾纹理的 PNG 图片作为背景。
   - CSS 动画: 通过 keyframes 设置两层云雾分别向左和向右极其缓慢地平移 (background-position-x 变化)，并使用 mix-blend-mode: screen 让它们交叠出暧昧的粉红雾气感。

2. 核心视觉中心: 天香美人图原画微缩版
   - 容器: 居中的卷轴画框，带有深沉的阴影 (box-shadow: 0 20px 50px rgba(0,0,0,0.8))，使其有悬浮感。
   - 滤镜: 画卷图片本身覆盖一层极其轻微的 sepia(棕褐色) 滤镜和粉色 tint 叠加，看起来古旧又色情。

3. 左侧监控区: 灵力欲海槽 (液体填充动画)
   - 容器结构: 一个高挑的、底部圆润的透明鼎炉轮廓 (SVG边框)。
   - 液体表现:
     鼎炉内部是一个相对定位的 div，高度通过 MVU 变量动态控制百分比 (height: XX%)。
     背景色使用线性渐变，从底部的深紫红过渡到顶部的桃花粉。
   - 液体波浪动画:
     利用伪元素 (::before, ::after) 绘制两个圆角巨大的半透明多边形，绝对定位在液体层顶部，不断进行 360 度无限旋转动画 (animation: spin linear infinite)，从而在视觉上形成液体表面翻滚、满溢的波浪效果。

4. 右侧监控区: 净欲刻印负荷 (环形进度条)
   - 结构: 基于 SVG 的圆形描边进度条 (stroke-dasharray 和 stroke-dashoffset)。
   - 常规状态: 进度条底色为暗灰，进度色为暗红，线条粗细适中。
   - 危险临界状态 (如负荷 > 80%):
     - 颜色突变: 进度条颜色通过 transition 变为刺眼的猩红色 (#FF0000)。
     - 心跳动画: 整个环形容器触发 keyframes 心跳缩放动画 (transform: scale(1) -> scale(1.05) -> scale(1))，频率与负荷度成正比。
     - 文本警告: 中间的负荷百分比数字字体加粗，并加上红色的文字模糊阴影 (text-shadow: 0 0 10px red)。

5. 底部阵法运转区: 实时数据流
   - 表现形式: 横向排列的数据卡片。卡片背景透明，仅有一条微弱的底部边框线。
   - 数值刷新动画: 当汲取效率或消耗速率的数值发生变化时，旧数字向上淡出 (transform: translateY(-10px), opacity: 0)，新数字从下方淡入 (transform: translateY(10px) -> 0, opacity: 1)。配合金粉色的颜色闪烁，展现出“采补”正在实时进行的动态感。

模块三: 【芥子】(小世界监控面板)
总体布局: 紧凑的 CSS Grid 四宫格布局 (grid-template-columns: 1fr 1fr; grid-template-rows: 1fr 1fr; gap: 15px)。

1. 宫格通用结构与材质 (四宫格的每一个卡片):
   - 容器边框: 使用带有点状纹理的细线 (border: 1px dotted rgba(212, 175, 55, 0.5))，并在四个角加上极其微小的青铜色金属护角 SVG 图标。
   - 遮罩滤镜 (核心淫靡感):
     每个卡片的底层是一个绝对定位的 div，设置了强烈的暗角效果 (box-shadow: inset 0 0 50px rgba(45, 26, 30, 0.9))，中间透出极其微弱的粉色光晕 (radial-gradient)。
   - 背景图像叠加:
     使用 CSS mix-blend-mode: multiply，将一张半透明的“肉色罗帐”肌理图覆盖在卡片背景上，让整个宫格看起来像是在窥视某个私密空间。

2. 宫格内部元素与动画:
   - 空间图标 (如教学楼、鸟居、床榻、刑架):
     使用极细线条的 SVG 绘制。图标颜色初始为淡墨灰。
     动画: 图标应用了一个极其缓慢的 SVG path 描边动画 (stroke-dashoffset 循环)，像是有什么东西在图标的线条上缓缓爬行。
   - 滞留人数与状态数据:
     字体使用极细的无衬线体，颜色为暗红色 (#C13256)。
     当某空间内有女性的发情度或臣服度发生剧烈变化时，该宫格的背景粉色光晕会触发一次时长为 2 秒的闪烁动画 (animation: pulse-glow)。

3. 交互展开 (展开名单列表):
   - 触发方式: 点击任意宫格，该宫格会通过 CSS transform: scale(1.05) 微微放大，同时同级的其他三个宫格通过 filter: blur(5px) 和 opacity: 0.3 虚化退入背景。
   - 列表呈现: 宫格内部滑出一个带纵向滚动条的名单容器。名单条目间的分割线是一条极细的红色渐变线 (linear-gradient: to right, transparent, red, transparent)。

模块四: 【群芳】(收藏品/女性详细信息 - 核心重构区)
总体布局: 经典的“左列表 - 右详情”结构 (Flexbox: 30% / 70%)。

1. 左侧栏: 绿头牌目录
   - 容器: 带有自定义深色内阴影的垂直滚动区。
   - 列表项 (绿头牌) 结构:
     - 形状: 顶部半圆、底部平齐的竖向长方形卡片 (border-radius: 20px 20px 0 0)。
     - 材质: 背景模拟陈旧的木牌质感 (linear-gradient + 木纹 noise 图)，名字字体为正楷，竖向排版。
   - 状态联动颜色:
     - 普通状态: 木色底，黑字。
     - 发情/高好感状态: 木牌底部会向上渗透出暗红色的渐变 (过渡动画 transition: background 0.5s)。
     - 彻底奴化状态: 整个木牌变为深红色，字体变为烫金色，并在牌子右上角附加一个微小的“印”字 SVG 标记。
   - 交互: 选中某个木牌时，木牌会有轻微的向右平移 (transform: translateX(10px))，并加上一层粉色外发光。

2. 右侧栏: 鼎炉秘档 (高密度数据面板)
   - 总体视觉: 像是一张被反复揉搓过的桃花笺展开，背景带有极其微弱的香艳暗纹。大量使用 CSS 折叠面板 (Accordion) 容纳高密度数据。
   - 数据区块通用设计 (折叠头部):
     - 头部背景: 暗红色的半透明渐变条。
     - 标题旁带有一个动态的“箭头” SVG，展开时旋转 90 度。
     - 边框: 只有下边框，使用双线 (border-bottom: 3px double #7B5C62)。

3. 关键数据组件的特殊前端表现:
   - 【奴化刻度】(臣服度进度条):
     - 表现: 不是常规的矩形进度条。背景槽是一个带有阴部轮廓隐喻的极简 SVG 图形。
     - 填充动画: 进度填充色为粘稠的乳白色与粉色交织的流体渐变 (linear-gradient)。满值时，进度条末端会触发一个极小的“液滴滴落”的 CSS keyframes 动画。
   - 【牝户秘档】(名器紧致度/爱液雷达):
     - 表现: 使用一组并排的半圆形仪表盘 (SVG half-circle gauge)。
     - 细节: 仪表盘的指针是一根红色的细针。当爱液泛滥度满值时，仪表盘的弧形轨道会闪烁水光质感的滤镜 (filter: drop-shadow(0 0 5px rgba(255,255,255,0.8)))。
   - 【榨取潜能】(烫金评级印章):
     - 结构: 一个绝对定位在面板右上角的圆形 div。
     - 材质: 背景为鲜红色，文字 (如 S, A, B) 为亮金色。
     - 滤镜与动画: 使用 CSS mix-blend-mode: multiply 融入背景纸张。加上一个永久的缓慢旋转缩放动画 (animation: stamp-breathe 4s ease-in-out infinite)，模拟一种压迫感。
   - 【绝顶弱点】(敏感点展示):
     - 表现: 文本描述旁边配有一个抽象的女性人体经络线稿 SVG。
     - 交互: 对应的敏感点位置会有一个红色的圆点闪烁。当 MVU 数据判定该弱点被触发时，红点会炸开一圈圈红色的波纹涟漪 (box-shadow 动画)。
